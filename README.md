## Part 1. 눈가리고 아웅하지 말기

### Chapter 1. 우리는 모두 거짓말쟁이

- #### 데이터가 필요한 이유
	- 창업가들은 종종 자신의 비전, 혹은 아이디어에 현혹되어 현실을 왜곡한다. 비전에 매몰되지 않고 자신의 현주소를 객관적으로 바라보기 위해 데이터르 비즈니스의 동반자로 삼아야한다.
- #### 최소한의 비용으로 테스트하라
	- 자신의 직감, 혹은 가설을 최소한의 비용으로 테스트하기 위해 MVP를 구상하라
